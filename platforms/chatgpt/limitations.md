# ChatGPT Limitations

This workflow improves consistency; it does not make generative image editing deterministic or guarantee commercial fidelity.

- An edit may extend beyond a selected or described area. Always compare the complete product against the original source after every render.
- Fine text, logos, stitching, woven patterns, earring findings, prongs, chain links, clasps, transparent stones, and reflective finishes require close human review.
- Color shown in an ordinary photo is not calibrated product color. Do not claim exact Hex, RGB, or production color accuracy without calibrated user input.
- Upload, generation, storage, and message limits vary by plan, workspace, location, model, and current service conditions.
- A reference image guides style but is not a strict segmentation mask. Reference products, props, typography, or color casts can leak into a result and must be rejected.
- The selection tool is not a precise mask, so changes may affect pixels outside the highlighted area.
- One failed targeted repair may be attempted from the original product source. If fidelity is still wrong, use Safe Run from the source; never continue repairing a generated result.
- If the bounded repair still fails, mark the image `MANUAL REVIEW` and use conventional retouching or reshoot the source.

For current interface behavior, see OpenAI's [ChatGPT Images guide](https://help.openai.com/en/articles/11084440-images-in-chatgpt).
